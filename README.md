## Métodos comunes en Python

| Método | Tipo | Descripción | Ejemplo |
|:-------|:------|:-------------|:---------|
| `.isdigit()` | String | Verifica si son dígitos | `"123".isdigit()` → `True` |
| `.strip()` | String | Elimina espacios extremos | `" hola ".strip()` → `"hola"` |
| `.replace()` | String | Reemplaza texto | `"abc".replace("b","x")` → `"axc"` |
| `.lower()` | String | Convierte a minúsculas | `"HOLA".lower()` → `"hola"` |
| `.split()` | String | Divide en lista | `"a b c".split()` → `["a","b","c"]` |
| `.append()` | Lista | Agrega al final | `[1,2].append(3)` → `[1,2,3]` |
| `.count()` | Ambos | Cuenta ocurrencias | `[1,2,1].count(1)` → `2` |
| `.pop()` | Lista | Elimina y retorna el último elemento | `[1,2,3].pop()` → `3`, lista `[1,2]` |
| `.clear()` | Lista | Vacía la lista | `[1,2,3].clear()` → `[]` |
